How to get started

  1. Download the latest build using the button above or clone the repository from GitHub.
  2. Upload the zip to WordPress via Plugins → Add New → Upload Plugin and activate it.
  3. Visit Tools → Github Plugin Installer and Updater, paste your repository URL, and configure optional tokens.
  4. Use the Managed Plugins table to connect existing plugins and trigger updates on demand.

Frequently asked questions

Can I use private repositories?

Yes. Generate a GitHub personal access token with the repo scope and add it to the plugin settings. The token stays in your WordPress database and is used only when downloading packages.

Does it work with multisite?

Absolutely. Network administrators can configure repositories once, then let individual sites run updates as needed.

How does the plugin update itself?

Point the self-update section to this repository and the plugin will surface update notices and provide one-click upgrades—no manual downloads required.
